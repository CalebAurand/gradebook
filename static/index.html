Student Routes

Unprotected Routes

/student-registration >>> allows a student to register in the database with a name, email and password

/student-login >>> allows a student to login and receive a JSON Web Token for later access to their class and grade information. The teachers will sign students up for their classes, and give them individual grades.

Protected Routes

/student-classes >> returns all the logged in student's classes they are enrolled in

/student-grades >> returns all the grades that a student has received
